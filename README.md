# Hi, I'm Alex

I am a programming hobbyist building practical projects with Python, databases, web APIs, automation, and now SwiftUI.

I like learning one layer at a time, understanding what the code is doing, and turning small experiments into working systems.

## What I'm Building

### X Bot Automation

A Python-powered automation system that:

* Fetches and stores quotes from external APIs
* Generates Brock Purdy-inspired motivational posts
* Publishes posts and replies through the X API
* Handles OAuth 2.0 token refresh
* Encrypts and stores mutable authentication data
* Tracks prompts, quotes, posts, users, media, and fetch activity in MySQL
* Runs automatically through scheduled PythonAnywhere tasks

This project started with JSON files and gradually evolved into a database-backed application with reusable helpers, logging, and automated workflows.

### Y-Madness

A Yahtzee-inspired dice game originally built with Python and Flask.

The project includes:

* Dice rolling and held-dice state
* Multiple rerolls
* Upper and lower scorecard logic
* Flask routes and templates
* Session-based game state
* Responsive Bootstrap UI

I am now learning Swift 6 and SwiftUI with the goal of rebuilding Y-Madness as a native iOS app.

### Euphoric Pixel

My personal Flask website and home for small applications, experiments, and future API endpoints.

The long-term goal is to connect my Python backend projects with native SwiftUI interfaces.
